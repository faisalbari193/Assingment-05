Question-1: What are the different ways to select an element in the DOM?

There are different ways to select element in dom.They are getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerText shows only visible text, innerHTML shows only all HTML tags and elements,textContent shows all text and hidden text also

Question-3: What is event delegation in the DOM?

Adding multiple event addEvenListener in child element is call event delegation

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept where a event starts from parent element to step by step child element and it again returns from child to parent element
